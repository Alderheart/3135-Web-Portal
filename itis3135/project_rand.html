Website Project Proposal

Project Overview:

  1. The purpose of my application is to be a collection of tools
    developed for administration purposes.
  2. The intended users of the website will only be for staff of
    Wayward Warriors, a game server.
  3. The content of the website would include a monthly compensation
    calculator and a way to log moderation actions and punishments.

Dynamic Functionality:

There will only be two interactive elements within this project proposal. The first is
the finance calculator, which will have buttons which are used to input values for a
final calculation. The second interactive element will be a front-end design for Google
Forms done with AJAX (this may be subject to change if it turns out to be too difficult).
An example for the calculator would be the Assignment #7, and an example for the form would
just be Google Forms itself. The pages these would occupy would be finance-calculator and
log-form respectively.
